# 🐄 QurbaniHat - Livestock Booking Platform

A modern livestock marketplace where users can explore animals for Qurbani such as cows and goats. Users can view details and place a booking after authentication.

## ✨ Key Features

- 🔐 **Authentication**: Email/password and Google social login using Better-Auth
- 🐄 **Animal Listings**: Browse cows and goats with detailed information
- 📋 **Booking System**: Book animals with a simple form (demo only)
- 🔍 **Sorting**: Sort animals by price (low to high / high to low)
- 👤 **User Profile**: View and update profile information
- 📱 **Responsive Design**: Fully responsive on mobile, tablet, and desktop
- ✨ **Animations**: Smooth animations using Animate.css
- 🔔 **Toast Notifications**: User-friendly notifications using React Hot Toast

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: JavaScript (JSX)
- **Styling**: Tailwind CSS
- **Authentication**: Better-Auth
- **Animations**: Animate.css
- **Icons**: React Icons
- **Notifications**: React Hot Toast
